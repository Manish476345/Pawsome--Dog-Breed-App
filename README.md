## Description

The Dog Breed Search App is a user-friendly Android application designed to help users explore different dog breeds. The app utilizes APIs to load a list of dog breeds along with their images and details. It offers features such as breed search, managing favorites, and the ability to share breeds with others. The app employs SQLite to seamlessly handle user preferences and maintain a local database for improved performance.

## Features

- Browse through a variety of dog breeds with images and details.
- Search for specific breeds using the search functionality.
- Mark your favorite breeds and manage them easily.
- Share interesting dog breeds with friends and family.
- Utilize SQLite for efficient storage and retrieval of user preferences.
- Enjoy a user-friendly and intuitive interface for a smooth experience.

## Installation

1. Clone the repository to your local machine using the following command:
git clone https://github.com/Manish476345/Pawsome--Dog-Breed-App.git

3. Open the project in Android Studio.

4. Build and run the app on an emulator or physical device.

## Usage

1. Upon opening the app, you'll be presented with a list of dog breeds.

2. Use the search bar to find specific breeds quickly.

3. Tap on a breed to view more details and an image.

4. click the "Favorites" button to add a breed to your favorites.

5. Access your favorite breeds through the favorites section.

6. Share breeds with others using the share button.

7. Enjoy exploring various dog breeds and managing your favorites!

## Technologies Used
